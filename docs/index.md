# Welcome to Zensical Documentation

This is a sample documentation site built with **Zensical** and deployed automatically to GitHub Pages using GitHub Actions.

## What is Zensical?

Zensical is a modern static site generator designed for technical writing and documentation. It's built by the creators of Material for MkDocs and combines the best of:

- **Static HTML generation** - Fast, secure, and easy to deploy
- **Markdown-first workflow** - Write in Markdown, get beautiful documentation
- **Automatic deployments** - Push to GitHub, see your docs live instantly
- **Material Design theme** - Beautiful, responsive design out of the box
